# 图书馆管理系统 java173

## 项目概述

图书馆管理系统是一款基于Java和Vue的前后端分离系统，运用SpringBoot框架进行开发，旨在提供便捷的图书管理、借阅流程以及用户权限管理功能。

## 技术栈

### 系统架构
- B/S架构
- MVC模式

### 运行环境
- 系统环境：Windows/Mac

### 开发环境
- 开发工具：IDEA
- JDK版本：1.8
- 项目构建：Maven
- 数据库：Mysql

### 技术组件
- Java
- SpringBoot
- Mybatis
- Vue

## 功能模块

### 管理员端
- 系统管理
- 读者管理
- 图书管理
- 借阅管理
- 公告管理

### 读者端
- 借书管理
- 借阅查看
- 公告查看

## 系统角色
- 管理员
- 读者
